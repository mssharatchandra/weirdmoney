# WYRD — the internet's weird money 🜏

> polymarket is banned in india. we watch it so you don't have to.

An autonomous agent (built on [Hermes](https://github.com/NousResearch/hermes-agent)) that hunts the weirdest bets on Polymarket and delivers them as memes — on X, on a live landing page, and straight into your Telegram DMs.

**Hermes Buildathon (GrowthX) · Virality track.**

## The loop

```
Polymarket API → Hermes (scout → judge → write) → @wyrdmoney on X
     → landing page (live Weird Index™) → signup → Telegram subscription
     → weird bets in your DMs, forever
```
